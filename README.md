# 🤖 AI Data Analyst Assistant

**AI Data Analyst Assistant** — это интеллектуальный веб-помощник для анализа данных, машинного обучения и визуализации. Загрузите CSV или Excel — и получите полный анализ, графики, ML‑модель и ответы на вопросы от ИИ.

Цель: Разработать веб-приложение для автоматического анализа данных, построения визуализаций, создания новых признаков (Feature Engineering), обучения моделей машинного обучения и получения ответов на вопросы о данных через ИИ (Groq LLM).



---

## ✨ Возможности

| Функция | Описание |
|---------|----------|
| 📊 **EDA** | Автоматический анализ: пропуски, дубликаты, выбросы, корреляции |
| 📈 **Визуализации** | Гистограммы, boxplot'ы, корреляционная матрица |
| 🤖 **AI‑вопросы** | Groq Llama 3.1 отвечает на вопросы о данных (без API‑ключа не требуется) |
| 🛠️ **Feature Engineering** | Извлечение из дат, комбинации признаков, one‑hot кодирование |
| 🤖 **Машинное обучение** | Регрессия / классификация (Random Forest), метрики, предсказания |
| 📋 **Отчёты** | Экспорт анализа в Markdown / HTML |
| 📥 **Экспорт** | Скачать очищенные данные (CSV), метрики (JSON), отчёты |
